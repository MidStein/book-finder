# Book Finder

Find that one book worth your time.

## Dependencies

- React
- TypeScript
- openlibrary api for data

## Features

- Find a book by it's title
- Find books by an author

## How to start the application

- `cd book-finder`
- `npm install`
- `npm run start`
- Visit on [localhost:5173](http://localhost:5173) on your web browser.
